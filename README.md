# Smart Budget Planner 💰

A Python command-line Smart Budget Planner built from scratch.

## Features
- Enter total budget and savings goal
- Auto-calculates spendable amount
- Add shopping items with priority levels (high/medium/low)
- Automatically allocates budget based on importance
- High priority → 50% of budget
- Medium priority → 30% of budget
- Low priority → 20% of budget
- Saves budget report to file

## How to Run
1. Install Python 3
2. Run: python budget.py

## Built With
- Python 3
- Lists and Dictionaries
- File handling
- Mathematical calculations
